# 📱 Como Adicionar Fotos pelo Celular

## ✨ Funcionalidade

Agora você pode:
- ✅ Adicionar fotos direto pelo celular (câmera ou galeria)
- ✅ As fotos aparecem no HTML
- ✅ Compartilhar direto no WhatsApp com as fotos já incorporadas
- ✅ As fotos ficam salvas no arquivo

## 📸 Como Usar

### 1️⃣ Abrir no Celular
1. Abra o arquivo `dia-dos-namorados (1).html` no navegador do seu celular
2. Pode ser pelo WhatsApp (compartilhar arquivo e abrir), ou por email, ou enviando para o celular de outra forma

### 2️⃣ Adicionar Foto de "Nossos Momentos"
1. Na tela "Nossos momentos", toque na área que diz "📸 Toque para adicionar foto"
2. Escolha: **Câmera** (tirar nova foto) ou **Galeria** (escolher foto já salva)
3. A foto aparece no mesmo instante! 🎉

### 3️⃣ Adicionar Fotos das Viagens
1. Na tela "Nossas viagens", toque em qualquer card de destino
2. Escolha a câmera ou galeria
3. A foto aparece no card dessa viagem
4. Repita para as outras viagens!

## 💾 Como Compartilhar no WhatsApp

Após adicionar as fotos:
1. Abra o navegador com o arquivo
2. Use a opção "Compartilhar" do seu navegador (⋮ → Compartilhar)
3. Escolha **WhatsApp**
4. Selecione o contato ou grupo
5. Envie! As fotos já estarão dentro do HTML

**Alternativa:** Você também pode simplesmente enviar o arquivo HTML diretamente pelo WhatsApp (como arquivo) após adicionar as fotos.

## 🎯 Dicas Importantes

- **As fotos são salvas automaticamente** no navegador do seu celular
- Se você fechar e reabrir, as fotos continuam lá
- Cada foto é convertida para um formato especial (Base64) que funciona sem internet
- Você pode tirar fotos novas quantas vezes quiser — basta tocar novamente para substituir

## ❓ Dúvidas?

- **A foto não apareceu?** Tente novamente, talvez tenha sido um erro da câmera
- **Quer mudar a foto?** Toque novamente no local e escolha outra
- **Quer remover a foto?** Infelizmente não dá para remover direto, mas você pode reabrir o arquivo original
